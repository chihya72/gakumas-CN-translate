PC (DMM) 安装教程

请先在下载页获取插件与资源，再按以下步骤操作。

1打开 1.插件本体.zip

压缩包内包含 gakumas-local/ 与 version.dll。

2解压到游戏安装目录

将 gakumas-local/ 与 version.dll 解压到与 gakumas.exe 同级的游戏根目录。

3准备最新资源包

打开 2.资源更新 的最新版资源压缩包，确认其中包含 local-files/。

4将资源覆盖到插件目录

打开刚解压的 gakumas-local/，将资源压缩包中的 local-files/ 解压进来并选择覆盖。

5汉化资源更新方法

如果需要对 PC 版汉化资源进行更新，既可以参照上面第3、第4步，下载最新资源包手动替换，也可以直接在游戏内更新。

游戏内更新方法：

在游戏彻底启动后，按下 Ctrl + U 组合键，出现如图所示的弹窗。在弹窗中完成最新资源包更新后，重启游戏即可体验最新汉化资源。

贴图替换资源更新

⚠️ 特别注意：贴图替换会改变游戏画面中的部分图片素材。若需要使用 MaaGakumasu 等依赖画面识别的自动化工具，请先关闭贴图替换功能，避免影响识别。

6打开插件控制台

等待游戏完全加载后，按下 Ctrl + U 组合键，打开 Gakumas Config GUI 插件控制台。确认基础设置中的 替换贴图 已勾选。

7展开资源设置

点击 资源设置 展开菜单。这里可以检查普通翻译资源，也可以检查贴图替换资源。

8检查贴图资源更新

勾选 从服务器检查贴图资源更新，确认 替换文件后删除下载缓存 保持开启，然后点击贴图 API 地址旁边的 检查更新。

9确认更新弹窗

出现贴图资源更新弹窗后，确认版本信息无误，点击 确定 开始下载。下载过程中请不要关闭游戏或插件控制台。

10等待下载完成

当插件日志中出现 Texture update completed. 时，说明贴图资源已经下载并安装完成。重新启动游戏后即可使用贴图替换效果。
