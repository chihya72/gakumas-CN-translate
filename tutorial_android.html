安卓 安装教程

⚠️ 特别注意：因为游戏本体 APK 是经过修补处理的，部分手机在安装时可能会提示"包含病毒"或"不安全"。
请放心安装！本汉化补丁仅修改游戏资源文件，不包含任何病毒、木马或恶意代码，不会窃取您的个人信息或造成任何安全问题。此类提示是因为修改了原始 APK 签名所致，属于正常现象。

安卓 需安装两个 APK：Localify 插件 APK + 修补版游戏本体 APK。请先在下载页获取这两个文件。

1安装 APK 并获取最新翻译资源

下载并安装 Localify 插件 APK 和 修补版游戏本体 APK（允许"未知来源应用"）。安装完成后，打开 Localify 插件，如图所示进行操作，获取最新翻译资源。

2等待资源下载并启动游戏

耐心等待最新资源下载完成后，点击右下角按钮启动游戏。

💡 提示：平时不需要更新资源的时候，可以直接点击游戏本体图标启动，无需通过插件。

3资源更新提示

如果有新资源发布，打开 Localify 插件时会有弹窗提示。点击"确定"会自动开始下载最新翻译资源。

4等待游戏加载翻译资源包

首次启动或更新资源后，游戏会加载翻译资源包，请耐心等待。加载完成后即可正常游玩。

5开始体验中文版

安装完成！现在你可以开始体验中文版的学园偶像大师了。祝您游戏愉快！🎉

贴图替换资源更新

⚠️ 特别注意：贴图替换会改变游戏画面中的部分图片素材。若需要使用 MaaGakumasu 等依赖画面识别的自动化工具，请先关闭贴图替换功能，避免影响识别。

6打开 Localify 插件

打开 Gakumas Localify 插件软件，确认基础设置中的 启用插件 与 替换贴图 已开启。

7展开资源设置

点击 资源设置 下方的 Details，展开资源更新相关选项。

8检查贴图资源更新

打开 从服务器检查贴图资源更新，确认 替换文件后删除下载缓存 保持开启，然后点击贴图 API 地址旁边的 检查更新。

9确认并等待下载完成

出现贴图资源更新弹窗后点击 确定，耐心等待资源下载完成。下载完成后启动游戏即可使用贴图替换效果。

10启动游戏查看效果

进入游戏后，如果标题画面、菜单等图片素材显示为中文，说明贴图替换功能已经生效。
